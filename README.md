# Pedra, Papel e Tesoura em JavaScript

Jogo clássico Pedra, Papel e Tesoura desenvolvido com HTML, CSS e JavaScript.



- Escolha entre pedra, papel ou tesoura
- O computador faz uma escolha aleatória
- O resultado do round é exibido na tela

Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

Objetivo do projeto

Projeto criado para praticar condicionais, funções, eventos e lógica básica de jogos, sem uso de frameworks.
